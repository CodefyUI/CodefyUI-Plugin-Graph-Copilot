# 附加檔案

你可以把**圖片、PDF、文字檔或程式碼**加進訊息 — 透過 **📎 按鈕**、**拖放**、或**貼上**。

## 各類型怎麼處理

- **圖片**會顯示成縮圖，並以 base64 **多模態**內容送給支援圖像的供應商（OpenAI / OpenAI-Codex / OpenRouter / Claude / 自訂）。若某後端或模型無法接受多模態內容，會優雅退化成一般串流錯誤，你可以重試。
- **文字與程式碼**檔會被擷取並以標註好的圍欄區塊**併入**提示。
- **PDF** 會被擷取成文字併入。解析使用**只在你真的附加 PDF 時才從 CDN 即時載入的 pdf.js** — 它從不打包進外掛，所以不會增加安裝體積。

## 標籤與限制

附加的檔案會以標籤列顯示在文字框上方（圖片縮圖或檔案圖示 + 名稱 + 大小，PDF 讀取中會有轉圈）。**只要還有附件在載入中，送出就會停用。** 每個檔案有大小上限，並附帶友善的錯誤訊息。
